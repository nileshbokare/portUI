SISA SMART - Compliance Management

SISA SMART is a single web based product to assist for performing the Compliance Management for various standards like ISO 27001, PCI DSS, FFIEC, SOX etc. This application having many modules like Compliance Management, Document management, Action Management and Setting side with many customization. SISA FSAQ – Facilitate Self-Assessment Questionnaires

SISA-FSAQ is to assist merchants and service providers in validating compliance with the PCI DSS number of Self -Assessment Questionnaires (SAQs) have been made available, each applicable to a specific payment scenario.
Benefits:
1. Helps organizations to automate their PCI DSS SAQs
2. Provides an online framework where Merchants/service providers can fill up their SAQ easily.
3. Built in work flow in the tool for approval/rejection of the project.
4. Feature to upload the evidences/reference files for each Questionnaire. Those evidences can be downloaded anytime and can be used for subsequent years in SAQ project
